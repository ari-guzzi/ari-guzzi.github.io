Game Design Projects

I completed a course called Tiny Games, spring 2024. A tiny game is a game with lots of technical limitations, usually they are mini games. I learned about the basics of game design and new software to create basic games. My favorite we used was Pico-8.

Bayou Bolt
For my final project in this class, I created a New Orleans themed 2D scrolling jumping platform game. I coded in Lua and used Pico-8 for its sprite and map editor to create the art.
